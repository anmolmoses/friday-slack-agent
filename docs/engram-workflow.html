How a Slack message flows through friday + engram

Friday's session manager owns every trigger — engram is a passive CLI/library it shells out to. The OpenAI key is used only for embeddings (vectorizing text on index & query). All reasoning — enrichment tagging, rerank, edge inference — shells out to the Claude subscription CLI (no key). Both halves are off unless ENGRAM_RECALL=1 / ENGRAM_CAPTURE=1, and every engram call fails soft.
